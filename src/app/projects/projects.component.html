Crypto Checker

Crypto Checker is a responsive web application built with Angular, designed to provide real-time cryptocurrency price tracking. Users can view up-to-date prices of various cryptocurrencies in either CAD or USD. The app features a sortable and paginated table for easy navigation, along with graphical charts for visualizing historical price trends. Users can switch currencies via a dropdown, and explore detailed data for individual cryptocurrencies. The clean and intuitive UI is built with Angular Material, making it user-friendly across different devices.

Tic-Tac-Toe PWA

The Tic-Tac-Toe Progressive Web App (PWA) is a modern, mobile-friendly web game built using Angular and Nebular UI components. It offers a clean and interactive game board for users to play the classic Tic-Tac-Toe game. The app is built as a PWA, allowing users to install it on their devices and use it offline. The game board is rendered dynamically, and the application provides a smooth user experience with responsive layouts. The theme is styled using Nebular's "Cosmic" theme, adding a vibrant and futuristic look to the UI.

Github: Tic-Tac-Toe
